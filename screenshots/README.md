# Screenshots

Please add the following screenshots before submitting:

1. **landing.png** - Landing page with audit form
2. **results.png** - Audit results page showing savings
3. **shareable.png** - Public shareable report page

## How to Take Screenshots:

1. Run `npm run dev`
2. Open http://localhost:3000
3. Take screenshots of:
   - Landing page
   - Complete an audit and screenshot results
   - Access shareable URL and screenshot
4. Save files as landing.png, results.png, shareable.png in this folder
5. Commit and push
